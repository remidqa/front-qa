Quality framework presentation

Overview

The quality framework is a solution to profesionalize quality and indistrualize your strategy. It is usable by all kind of roles, such as QA, PO or developpers, even your cicd pipelines, and do not require any technical knowledge.

Concretely, one action on the framework will :
  • Get tests for the required application (functional, API, load tests,... even all at once)
  • Execute the tests and wait for the result
  • Get and analyse the report
  • Alert you by chat in case of failure
  • Store the report in a database
  • Save the metrics for a graph solution
  • Update results in the test management solution

Available features - not exhaustive, there is way too many possibilities :D
  • Manage dataset by environments
  • Execute tests for application, dependecies and/or consumers
  • Define test strategy by environment (like run "load tests" only in preproduction)
  • Connect to a secret management solution
  • Use from a chat application (like Slack)
  • ...

The situation

In all companies, a lot of softwares, applications, API,... are used to perform the quality strategy.

And all actors of the quality strategy needs to navigate, manually, between all those tools to complete quality actions.

Let's have a look at all applications and users, the posibilities are enormous and things can quickly can become laborious for everyone.

  • APPLICATIONS
    • For testing
      • Functional Tests
      • API tests
      • Unit Tests
      • ...
    • For repoting
      • Monitoring
      • Chat alerts
      • History in database
      • ...
    • For managing
      • Bug tracking
      • Test book management
      • Sprint validation
      • ...
  • USERS
    • QA
    • PO
    • Developper
    • Cicd pipeline
    • ...
Risks
Consequences
  • Increase the number of errors due to manual actions
  • Lower the efficiency of the users because they need time and focus on repetitive tasks
  • Lower the attractivity and reliability of your quality strategy
  • Limit yourself and exclude features from your quality strategy to gain time and labor
